#Sentiment Analysis of Flipkart Product Reviews

Analyzed e-commerce reviews to classify sentiments as a group project, with my focus on NLP implementation and model building.

##Overview
- **Data**: 180,000+ reviews.
- **Team Context**: Developed with classmates for a course project (Jan-May 2023).
- **My Role**: Implemented the sentiment analysis pipeline using TextBlob, trained ML models and generated visualizations (Word Cloud).
- **Tools**: Python, TextBlob, Matplotlib, WordCloud.
